## 🧩 The Ultimate Library Union (50+ Tools)

### 1. Agentic Frameworks & Frameworks
Modern tools that treat the browser as an environment for LLM agents.
- **Stagehand**: High-level agentic browsing framework.
- **Browser-Use**: Autonomous browser tool-use for LLMs.
- **MultiOn**: The pioneer in autonomous agentic browsing.
- **Skyvern**: Vision-based browser automation workflows.
- **Kadoa**: Precision vision-extraction API.

### 2. AI-Native & RAG Extraction
Tools designed to convert the messy web into clean data for AI.
- **Crawl4AI**: The gold standard for web-to-markdown conversion.
- **Firecrawl**: Managed crawler for LLM data ingestion.
- **ScrapeGraphAI**: Prompt-driven dynamic parsing graphs.
- **Marker-PDF**: High-fidelity PDF-to-Markdown (OCR).
- **Goose3**: Article and metadata cleaner.

### 3. Modern Stealth & Steering
Bypass advanced anti-bot systems like Cloudflare, Akamai, and DataDome.
- **nodriver**: High-speed, async bot-bypass (successor to UC).
- **curl_cffi**: Industry standard for TLS/JA3 impersonation.
- **Camoufox**: Python-native anti-detect browser.
- **SeleniumBase**: The most feature-rich Selenium stealth wrapper.
- **DrissionPage**: Smooth hybrid requests/browser engine.

### 4. Specialized & Domain Tools
- **yt-dlp**: Universal media and metadata extraction.
- **Trafilatura**: Advanced news and blog cleaning.
- **Extract**: JSON-LD, Microdata, and OpenGraph ingestion.
- **Pywebcopy**: Full site cloning and "grabbing".

### 5. Data Cleaning & Production Utilities
- **Dateparser**: Natural language date conversion.
- **Price-parser**: Ecommerce price and currency extraction.
- **Tenacity**: Production-grade retrying logic.
- **fake-useragent**: Realistic browser identity rotation.

## 🚦 Installation

We recommend **modular installation** to avoid industry-wide version conflicts between Agentic and Automation stacks:

```bash
# Ultimate AI/Agent Stack
pip install .[agentic,ai,cleaning]

# Pro Automation Stack
pip install .[modern-core,stealth,specialized]

# Foundational Basics
pip install .[foundational]
```
