# Contributing to Awesome Python Scraping Toolkit

We welcome contributions! Whether you're adding a new library, fixing a bug in an example, or improving documentation, here's how you can help.

## How to Contribute

1.  **Search first**: Check if the library is already in the list or has been discussed in issues/PRs.
2.  **Fork & Branch**: Create a feature branch for your additions.
3.  **Adhere to Format**:
    - Keep descriptions concise.
    - Categorize correctly.
    - Mention if it's AI-native or has specific prerequisites.
4.  **Add examples**: If adding a major library, consider adding a 10-20 line "Hello World" in the `examples/` folder.
5.  **Submit PR**: Document why the library is a valuable addition to the ecosystem.

## Criteria for Inclusion
- Must be a Python library or a CLI tool with a Python wrapper.
- Must be related to web scraping, crawling, site-grabbing, or parsing.
- Preference given to active, well-documented projects.
